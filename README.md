# Semantic Profile Portfolio (WCAG Compliant)

A modern, responsive, and **accessible personal profile portfolio website** built using **semantic HTML5 and CSS**, following **WCAG 2.1 accessibility guidelines** and best frontend practices.

This project demonstrates clean UI design, structured content, and accessibility-first development — suitable for professional portfolios, internships, and frontend evaluations.
